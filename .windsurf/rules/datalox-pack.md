# Datalox Pack Rules

This repo is a Datalox pack.

Read:

1. `.datalox/manifest.json`
2. `.datalox/config.json`
3. `DATALOX.md`

Then use the loop:

`detect -> use -> patch -> lint`

Keep these files current:

- `.datalox/index.md`
- `.datalox/log.md`
- `.datalox/lint.md`

Do not replace native Windsurf or Cascade skills. Datalox is additive.
