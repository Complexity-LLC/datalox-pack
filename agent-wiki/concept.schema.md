# Agent Wiki Concept Page Schema

Concept pages explain reusable domain ideas that sit behind one or more patterns.

## File Location

- Store concept pages in `agent-wiki/concepts/`

## Recommended Frontmatter

```yaml
---
type: concept
title: Human-readable concept title
workflow: workflow_id
status: active | stale | superseded
related: []
sources: []
author: name
updated: 2026-04-12T10:31:16.852Z
review_after: 2026-07-12
---
```

## Recommended Sections

### Definition

What the concept means in one clear paragraph.

### Why It Matters

Why the agent should care about this concept during work.

### Examples

Concrete representative cases.

### Related

Wiki-relative paths to associated patterns, questions, or comparisons.
