# Review Ambiguous Viability Gate

## When To Use This

Use this doc when live/dead separation is unclear during viability gate review.

## Steps

1. Inspect whether the dim dead tail is distinct from the live cluster shoulder.
2. Verify FSC and SSC cleanup before adjusting the gate.
3. Keep the live boundary conservative if the separation remains ambiguous.

## Judgment Patterns

- Treat overlap after cleanup as a review problem, not an automatic gate shift.
- Prefer the boundary that preserves an auditable explanation over a visually aggressive correction.

## Escalation Rules

- Escalate when the gate change would materially alter the reported live population.
