# Gemini Instructions

Use this pack with the loop:

`detect -> use -> record -> promote -> lint`

Read:

1. `.datalox/manifest.json`
2. `.datalox/config.json`
3. selected skill in `skills/`
4. linked notes in `agent-wiki/notes/`

Promote repeated reusable knowledge into notes first.
Promote repeated reusable procedures into skills.
